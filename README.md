minitwitter
===========

MiniTwitter by dfs



REQUIREMENTS
===========

Yii-1.1.14


INSTALLATION
===========

1. Install Yii 1.1.14
2. Extract all files to your Yii root directory
3. Change database params in config: /protected/config/main.php
4. Change admin e-mail: /protected/components/Controller.php
